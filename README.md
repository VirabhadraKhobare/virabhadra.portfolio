# virabhadra.portfolio

Professional full-stack developer portfolio for **Virabhadra Khobare** built with a modern React + Vite frontend and a scalable Node.js + Express backend.

---

## 🚀 Live Demo

🌐 Frontend:  
https://virabhadra-portfolio-frontend.vercel.app/

---

# 📌 Features

- Modern responsive portfolio UI
- Animated user experience using Framer Motion
- Admin dashboard for content management
- Blog management system
- Secure JWT authentication
- MongoDB database integration
- Contact form with email support
- Analytics and portfolio statistics
- REST API architecture
- Production-ready deployment setup

---

# 🛠️ Tech Stack

## Frontend

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Axios

## Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Nodemailer
